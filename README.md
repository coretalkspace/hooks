# Misskey hooks

Cloudflare Worker with Durable Objects: WebSocket rooms per username and a POST webhook that publishes events into those rooms. Used with the [Fediverse Notifications](https://github.com/coretalkspace/plugin-notifications) Misskey plugin.

The default fediverse portal this stack targets is **[coretalk.space](https://coretalk.space)** (worker host pattern: `hooks.coretalk.space`).

## Related

- Misskey plugin: [coretalkspace/plugin-notifications](https://github.com/coretalkspace/plugin-notifications) — install from [raw `notifications.aiscript`](https://raw.githubusercontent.com/coretalkspace/plugin-notifications/main/notifications.aiscript).
